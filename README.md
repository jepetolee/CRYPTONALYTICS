# CRYPTONALYTICS
## 인공지능이 비트코인을 투자할 수 있을까?
### 개요
#### 필자는 인공지능 기술을 이용한 가상화폐 선물 거래가 가능한지가 궁금했고, 직접 강화학습을 활용해서 인공지능이 가상화폐 투자에서 강세를 보이는 지 가상환경, 가상 거래 환경, 실제 거래에서 수익을 올리는 지에 대해 각각 실험을 진행해보았다.<br>
#### 결론을 우선 말하자면, 반쪽짜리 성공이었다.
